# Project instructions for Claude Code

This is Miriam Hanna's personal portfolio site (miriamhanna.com), rebuilt from a Neocities
HTML/CSS/JS site into React. The full architecture/blueprint for this rebuild lives outside
this repo (it was produced and approved before any code here was written); this file only
holds standing rules for working in this repo specifically.

## README maintenance (required)

Whenever a change affects setup steps, dependencies, folder structure, environment variables,
or deployment, update `README.md` in the same change. Do not leave it for a separate pass —
the README is a living document, not a one-time file.

## Do not commit

- The original Neocities export (HTML/CSS/JS source) — this repo starts fresh; only extracted
  content and freshly written React/TypeScript code belongs here.
- Any email address, `mailto:` link, or other direct contact-exposing content — the contact
  method is a form (Formspree) by design; no email address appears anywhere in this repo.
- Large raw/uncompressed video source files — compress before adding (see README/deploy notes).

## Licensing

No open-source license is used intentionally. The repo is public for portfolio/demonstration
purposes only; do not add an OSI license file without being asked.
